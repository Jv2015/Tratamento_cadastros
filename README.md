# **TRATAMENTO DE DADOSCOM BASE EM PERFIL**

O projeto foi desenvolvido com o objetivo de tratar e integrar duas bases de dados distintas. Utilizando Python, foi aplicado um método de separação por queries, visando estruturar os dados em um modelo estrela, que organiza as informações em tabelas fato e dimensão para facilitar a análise. A partir dessa base estruturada, foi criado um dashboard para explorar o perfil demográfico dos usuários, com ênfase em localização, características comportamentais e outros atributos relevantes presentes nos dados.
